Codex 配置 API Key 教程：中转站注册、密钥创建与覆盖配置完整指南

本教程面向国内用户，主要介绍如何为 Codex CLI、Codex App、VS Code Codex 插件 配置 API Key，并通过中转站完成覆盖配置。

01. Codex 中转站地址

本文使用的 dkai-codex 中转站地址如下，该站点主要面向 Codex、Claude Code 等编程工具调用场景。

dkai-codex 中转站注册地址：
https://codex.dakeai.cc/register?promo=DKAI

如果你还有 OpenClaw 或其他支持第三方自定义 API 接入的 Agent、AI 程序调用需求，也可以使用 dkapi。

dkapi 通用 API 中转站：
https://api.dakeai.cc/

dkapi 支持多种常见兼容协议，一个 Key 可以调用多款大模型。使用时通常只需要根据兼容协议替换 URL 和 Key 即可。

Codex App 配置画图 API 教程：
https://dk82.com/714.html
注册后系统通常会赠送一定余额，可用于免费体验。建议先测试是否满足自身需求，再决定是否付费。

02. 使用中转站前需要准备什么？

中转站使用主要分为两个关键步骤：

  1. 安装对应工具平台，例如 Codex App、Codex CLI、VS Code Codex 插件或 Claude Code。
  2. 执行覆盖配置，让工具调用新的 API Key 和中转接口地址。

如果你已经安装过相关工具，可以跳过安装步骤，直接进行覆盖配置。

Codex Desktop App 下载安装与配置教程：
https://dk82.com/635.html
VS Code Codex 插件安装教程：
https://zhuanlan.zhihu.com/p/2035752399952950892

03. 注册并创建 API 密钥

注册并登录中转站后，进入 API 密钥页面，创建一个新的 API Key。

  1. 打开 API 密钥页面。
  2. 点击创建密钥。
  3. 填写密钥名称。
  4. 选择对应分组。
  5. 保存并生成 API Key。

04. 创建密钥时分组怎么选？

密钥名称可以自定义填写，但 分组必须选择正确。不同分组对应不同模型和工具。

  • Codex 分组：适合 Codex CLI、Codex App、VS Code Codex 插件等工具。
  • Claude 分组：适合 Claude Code 相关工具。

如果你要配置的是 Codex，建议选择 Codex 分组。

05. 使用密钥并获取覆盖配置脚本

以 Codex 为例，选择 Codex 分组填写信息并创建密钥。随后在密钥列表中找到刚刚创建的密钥，点击右侧的 使用密钥。

进入使用密钥页面后，根据你的操作系统选择对应方法。

  • 如果未安装工具，先执行安装步骤。
  • 如果已经安装工具，可以直接执行覆盖配置。

06. 执行覆盖配置脚本

覆盖配置是整个教程中最关键的一步。安装完成后，如果不覆盖配置，工具通常无法正常调用接口。

  1. 在“使用密钥”页面找到覆盖配置脚本。
  2. 点击复制脚本。
  3. 打开终端、CMD 或 PowerShell。
  4. 粘贴脚本并执行。
  5. 等待提示配置成功。
  6. 重启 Codex CLI、Codex App 或 VS Code Codex 插件。

07. 启动 Codex CLI 测试

安装并覆盖配置完成后，可以启动 Codex CLI 进行测试。

  • macOS 用户：打开 Terminal。
  • Linux 用户：打开终端。
  • Windows 用户：打开 CMD 或 PowerShell。

然后输入：

codex

按回车后即可启动 Codex CLI。第一次进入时，可能会出现几个提示，一般直接按回车即可。

启动成功后，可以直接输入你的需求，例如：

帮我写一个简单的个人博客首页

或者：

帮我检查这个 Python 脚本为什么报错

08. 配置完成后可以做什么？

配置完成后，你可以使用 Codex 完成很多编程相关任务，例如：

  • 生成网页代码
  • 修改已有项目
  • 解释代码逻辑
  • 排查报错原因
  • 重构代码结构
  • 编写脚本和自动化工具
  • 辅助完成网站、小程序、桌面应用等项目
如果你不知道怎么使用，也可以直接问 Codex，例如：“我完全不会写代码，想做一个简单的网站，你能一步步教我吗？”

09. 总结流程

完整流程如下：

注册中转站 → 创建 API 密钥 → 选择 Codex 分组 → 点击使用密钥 → 复制覆盖配置脚本 → 终端执行 → 重启工具 → 开始使用

完成以上步骤后，Codex CLI、Codex App 或 VS Code Codex 插件就可以正常调用中转 API 使用了。

本页面仅用于 Codex API Key 配置与中转站设置教程整理，请以实际平台说明和官方工具提示为准。
